0x01 简介

kimi-cli 是 kimi 出的辅助编程工具。

名字叫“cli”，潜意识以为是 LLM 融合 cli，生成复杂的 linux 命令组合，例如这种：

find . -type f | xargs ls -lah | awk -F..

所以 repo 想火，起个能火的名字很重要，最好外国人也能懂。不信你看 open-r1..

实际功能和 OpenHands 类似，肉眼扫了下源码：

LLM 有明确的 file 工具列表，“极大概率不会”生成危险的 rm
内部实现简洁，例如想增加 git 自动提交能力，比较好改
二次元成分高，作者必然大佬，“El Psy Kongroo”
