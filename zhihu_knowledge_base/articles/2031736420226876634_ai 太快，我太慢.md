# ai 太快，我太慢

**作者**: 东川路第一伊蕾娜​Open and open again

**原文链接**: https://zhuanlan.zhihu.com/p/2031736420226876634

---

“ai时代，只要你学的够慢，你就可以不用学。” ， ”auto research is all you need “， “gpt5.5” ，“deepseek v4 pro”......

见证了LLM scale 阶段，到 推理能力 的post-training 阶段，再到现在的long context + 1T基模 才能成为入场券的阶段。这些事情仅仅发生在2年以内，每一个月都觉得被ai时代的浪潮抛到后面。

主播最近几个月持续性follow，开了claude 会员，又有了几千claude/月的api；research 尝试全链路采用gpt pro + 项目结合，持续deep research 研究arxiv；不断刷X，知乎和小红书 去搜索信息；不断看油管和B站看访谈，和新概念的解释；用openclaw 每天定时推送各种概念复习......

虽然主播已经竭尽全力去压缩各种信息，但还是受限于学习能力和精力，实在难以长期兼顾。一方面是持续接触更多新概念和思路的兴奋，另一方面是自己精力有限，个人也是要有自己的生活的，所以感觉实在学不完。

有意思的是，主播发现ai 时代下，任务可以划分为：

环境确定下的改进：几乎完全靠ai，比如某一个trick，消融 等等，可以通过claude code plan+codex review 全链路解决，只要自己给的prompt 合理，就可以做到；
搭建环境：部分依赖于自己 ，claude code 更多只能做助手，codex 可以帮忙review，但动不动就超时之类的都不行；
idea设计：大部分依赖自己，gpt 和gemini可以帮忙拓展context（比如问它是否有人实现过，可以拿到一个比较准的文献参考）
实验设定：几乎完全依赖自己，需要利用实验思维去敲定怎么做；
基础知识储备：完全靠自己，当然如果问答解决问题也没问题；
领域选择：完全靠自己，需要各种访谈与talk才能懂得。

我们更应该把精力投入到完全靠自己的领域上，这样才能让有限的精力发挥更大的价值。但不幸的是，现实世界往往会有持续的压力，比如submit的ddl，比如考试等等。我们持续性把精力投入到了1，2，3 完成上，而忽视了4，5，6这种需要高精力支撑的部分。

主播最近也开始尝试一些新思路：

1.每天晚上和周末单独拿出一点时间，去做5和6。

2.每天早上拿出时间做 4 ，统领1，2。

3.下午半走神的干活... ...
