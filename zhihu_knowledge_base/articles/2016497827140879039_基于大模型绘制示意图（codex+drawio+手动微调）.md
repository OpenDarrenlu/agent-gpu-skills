# 基于大模型绘制示意图（codex+drawio+手动微调）

**作者**: Arsmart​上海交通大学 计算机科学技术博士在读

**原文链接**: https://zhuanlan.zhihu.com/p/2016497827140879039

大模型生成图片的困扰常常在于不方便微调细节，那么解决思路是让大模型修改可微调的文件格式，和人类实时交互。在这个思路的指引下，我使用codex+drawio，绘制了某论文的示意图，该思路显然可以推广到PPT制作，Word书写等情景。

具体案例如下：

大模型生成的png，难以手动调整
codex开启的动态修改界面。

代码仓库如下：

https://github.com/ziyuhuang123/auto_drawio

如何使用：

打开codex，告知当前需要修改的.drawio文件的位置，然后在浏览器里打开http://127.0.0.1:4318/便可看到此界面，大模型的每次修改都会自动体现在这里，然后人也可以手动进行微调。比如prompt可以是：我现在希望修改的文件是：C:\文件备份\new_report\0313\未命名绘图.drawio 帮我连接服务器：http://127.0.0.1:4318/
