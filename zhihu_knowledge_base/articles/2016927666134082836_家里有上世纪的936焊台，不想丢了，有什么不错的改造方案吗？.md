二、软件设计方案

软件层面，使用FreeRTOS实时操作系统

工作原理：测温→PID算功率→过零触发→控制加热
烧录说明：USB可直连，无需专用烧录器

-5000 ~ +5000    p->intEg += err;

// 微分    
const int16_t diff = err - p->lastErr;      
// PID输出    
int32_t out = kp*err + ki*intEg + kd*diff;   out /= 200;

0-99    if (out < 0) out = 0;
