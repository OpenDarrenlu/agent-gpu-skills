# 清华全新可训练稀疏注意力SLA：稀疏与线性结合，提速26倍！

近期，稀疏注意力证明了其在Video Generation任务上的有效性，比如SpargeAttention、SVG等。但注意力真的非常稀疏吗？答案可能出乎意料。例如，在30K这种中等长度的序列上，注意力的稀疏度可能只有50%，如果强行拉高注意力的稀疏度，会导致严重的准确度下降。近日，清华大学朱军团队发现Diffusion模型中的注意力图可以普遍地别拆解为两部分：一部分极度稀疏、另一部分极度低秩。根据这个重要观察，团队提出了SLA (Sparse-Linear Attention),将稀疏注意力与线形注意力在层内进行了完美融合，即使在30K这种中等序列长度上，SLA也可以实现95%的无损注意力稀疏度。并且团队还将SLA实现在了SageAttention的基础上（SageSLA），实现了最高26倍的注意力加速。







论文标题：SLA: Beyond Sparsity in Diffusion Transformers via Fine-Tunable Sparse–Linear Attention

论文链接：https://arxiv.org/abs/2509.24006

开源代码：https://github.com/thu-ml/SLA




代码已全部开源，在https://github.com/thu-ml/SLA中，包括SLA（Sparse-Linear Attention）的前向与反向过程，还包含26倍加速的SageSLA (SageAttention + SLA)!

使用起来还非常简单，只需用两行代码替换原始SDPA注意力即可。




难点：注意力并不非常稀疏

以Wan-2.1-1.3B-480P的模型为例，其序列长度为30K。即便使用理想条件下最优的稀疏注意力，可以发现在稀疏度在50%左右时，注意力的准确度还可以接受，但如果强行拉高至90%则会产生难以接受的注意力误差（L1 Error > 33%）。







重要发现：注意力图的完美性质，极度稀疏与极度低秩

团队发现，普遍地，在Diffusion Model中注意力图矩阵（P）可以分拆为两部分：

极度稀疏部分：最大的8%数量的值（Rank = 6230）；
极度低秩部分：其余所有92%数量值（Rank = 9）。




这一重要发现完美建议了可以对“极度稀疏部分”采用稀疏注意力，对“极度低秩部分”采用低秩（线形）注意力。




方法 - SLA（Sparse–Linear Attention）

SLA（Sparse–Linear Attention）是一种可训练的稀疏注意力，将稀疏注意力与线性注意力完美融合：




对于注意力图（P）中较大的值对应的计算，SLA采用稀疏注意力，对于其他运算则采用线性注意力。

SLA（Sparse–Linear Attention）实现了高效的前向和反向过程，使用时，只需使用SLA替换原始的注意力，然后仅需用训练数据微调模型少量步数即可直接用于推理过程。
