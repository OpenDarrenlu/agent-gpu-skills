社区发展

在3月，TVM加入了Apache软件基金会，成为了一个孵化项目。(新闻链接).

我们欢迎 Apache 孵化器的成员 Markus Weimer (@markusweimer), Sebastian Schelter (@sscdotopen), Byung-Gon Chun (@bgchun), Henry Saputra (@hsaputra), Timothy Chen (@tnachen), Furkan KAMACI (@kamaci)。他们将会帮助TVM项目完成孵化的完整流程。

同时，社区加入了新的committer： Morita Kazutaka (@kazum)， Yao Wang (@kevinthesun)，和新的reviewer： Xingjian Shi (@sxjscience), Wei Chen (@wweic), Sergei Grechanik (@sgrechanik-h)。

社区讨论论坛（ discuss.tvm.ai ）继续健康成长，在3月累计页面浏览量4万8千次，用户访问量1千7百次。

新特性和改进

社区在继续推进新的可微分函数式中间表示 Relay，新的特性包括高阶自动微分，pass管理器，更多语言特性支持和更广的算子/前端覆盖（ 如编译带控制流的Tensorflow模型 #2830)。我们对新IR的虚拟机设计开展了讨论(#2810 #2915)，并且有了一份草稿实现(#2889)。

在底层编译器方面，我们正在重构整数集分析(#2588)，目前完成了常量整数界限分析，模集合分析（modular set analysis）和基于重写的表达式化简器。

其他的改进包括更好的CPU指令生成，新的优化pass，CI/编译稳定性的提升和其他bug修复。细节见下表。
