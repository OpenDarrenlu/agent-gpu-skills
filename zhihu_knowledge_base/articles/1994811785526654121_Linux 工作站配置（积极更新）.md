fzf

https://github.com/junegunn/fzf

超级强的筛选工具。其作用之一是命令搜索工具、取代ctrl+R的神器！速度快、检索方便、非常好用。

ctrl+R 使用效果：


安装方法：
ubuntu24下安装，apt安装的版本太老了，得使用brew：

brew install fzf
echo 'eval "$(fzf --bash)"' > ~/.bashrc
Zoxide

https://github.com/ajeetdsouza/zoxide

这个工具可以帮你快速跳转到之前使用过的路径。

比起autojump，zoxide和fzf的集成更好，且有着rust运行快的优势。所以笔者在2026年开始强烈推荐用Zoxide啦！

# linux, bash, 环境下安装。记得先装fzf。
curl -sSfL https://raw.githubusercontent.com/ajeetdsouza/zoxide/main/install.sh | sh
eval "$(zoxide init bash)"
# 添加下面这个，用来在当前目录下快速跳转任意子路经
alias zl='cd "$(find . -type d | grep -v "^\.$" | fzf --height 40% --reverse --preview "ls -la {}" --preview-window right:60% --select-1 --exit-0)"'

tmux

懂的都懂！

贴一下我的.tmux配置（~/.tmux.conf ）

# 推荐的终端类型（tmux 3.1+ 原生支持）
set -g default-terminal "tmux-256color"
# 启用真色支持（可选，适配 vim 真色）
set -ga terminal-overrides ',*:RGB'

# 窗口/面板编号从 1 开始（无问题，保留）
set-option -g base-index 1
set-window-option -g pane-base-index 1

# 重新绑定前缀键循环面板（规范写法，保留核心功能）
unbind C-b  # 规范写法：小写 b，和 ^B 等价
bind C-b select-pane -t :.+

# 重载配置的快捷键（保留）
bind-key r source-file ~/.tmux.conf \; display "Config reloaded!"

set-option -g mouse on

# 修复滚动问题（更兼容的写法）
# 仅针对 xterm 系终端，且不强制禁用 smcup/rmcup
set -g terminal-overrides 'xterm-256color:smcup@:rmcup@,alacritty:smcup@:rmcup@'
# 启用 tmux 内置滚动缓冲区（增强滚动体验）
set -g history-limit 10000

# 智能面板切换（修复 C-l 绑定冲突）
# 全局绑定 C-h/j/k/l 切换面板，且兼容 vim 拆分窗口
bind -n C-h run "([ $(tmux display-message -p '#{pane_current_command}') = vim ] && tmux send-keys C-h) || tmux select-pane -L"
bind -n C-j run "([ $(tmux display-message -p '#{pane_current_command}') = vim ] && tmux send-keys C-j) || tmux select-pane -D"
bind -n C-k run "([ $(tmux display-message -p '#{pane_current_command}') = vim ] && tmux send-keys C-k) || tmux select-pane -U"
bind -n C-l run "([ $(tmux display-message -p '#{pane_current_command}') = vim ] && tmux send-keys C-l) || tmux select-pane -R"
bind -n C-\\ run "([ $(tmux display-message -p '#{pane_current_command}') = vim ] && tmux send-keys 'C-\\') || tmux select-pane -l"


# 1. 开启tmux和系统剪贴板的联动（关键）
set -g set-clipboard on
# 2. 复制模式使用vim键位，绑定y键复制到Mac剪贴板
set-window-option -g mode-keys vi
bind-key -T copy-mode-vi v send -X begin-selection
bind-key -T copy-mode-vi V send -X select-line
bind-key -T copy-mode-vi y send -X copy-pipe-and-cancel "xclip -selection clipboard"
bind-key -T copy-mode-vi MouseDragEnd1Pane send -X copy-pipe-and-cancel "xclip -selection clipboard -i"

# ==================== Mac专属剪贴板配置 ====================
# Mac先装一个xquartz, 服务器上装一个xclip，然后才能使用下面的x11转发

AI
tmuxai

https://tmuxai.dev/

很方便的AI辅助终端，agent像一个同事一样，在你旁边帮你解决问题。

优点：有两个框，一个用来对话，一个用来运行真正的命令。agent能直接运行命令、观察结果，而你也可以非常简单直接地修改命令、新增命令，实现和agent的共舞。

缺点：2026.02.13，tmuxai还有成长空间，目前消耗token太暴力了。给作者提了个小小的建议，看看咋说：https://github.com/alvinunreal/tmuxai/issues/150
