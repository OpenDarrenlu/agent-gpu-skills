近日，清华大学朱军教授团队联合生数科技开源了视频生成加速框架 TurboDiffusion，在单张 RTX 5090显卡上即可实现端到端100–200 倍的视频生成加速。当前，视频生成已成为 AI 大模型最重要的应用方向之一，切需求量不断扩大，但其生成效率一直难以满足实际需求。TurboDiffusion 的推出显著提升了视频生成的效率和可用性。







TurboDiffusion的用法

TurboDiffusion使用起来非常方便，高效推理代码以及TurboWan2.2/2.1的图生视频、文生视频的模型参数（Checkpoints）都已开源，可一键生成。

具体来说，只需安装TurboDiffusion仓库（https://github.com/thu-ml/TurboDiffusion）中的Python Package，并下载对应的模型Checkpoints，就可以直接调用仓库中的推理命令来生成视频。




TurboDiffusion的主要技术

TurboDiffusion目前主要包含以下几种技术来加速扩散模型：

1. TurboDiffusion 使用 SageAttention （https://github.com/thu-ml/SageAttention）进行低比特量化注意力加速。

2. TurboDiffusion 使用 Sparse-Linear Attention（SLA）（https://github.com/thu-ml/SLA）实现稀疏注意力加速。由于稀疏计算与低比特 Tensor Core 加速是正交的，SLA 可以构建在 SageAttention 之上，在推理过程中进一步获得数倍的额外加速。

3. TurboDiffusion 引入 rCM （https://github.com/NVlabs/rcm）进行步数蒸馏加速，该方法目前属于该方向的先进方案之一。

4. 最后，TurboDiffusion 对 Linear 层采用 W8A8 量化进行加速，使用 INT8 数据类型，并在 128×128 的块粒度上进行分块量化。




TurboDiffusion的效果

总的来说，TurboDiffusion在单张 RTX 5090显卡上即可实现端到端100–200 倍的几乎无损的视频生成加速。Demo展示:

TurboWan2.2-I2V-14B-720P

对于Wan2.2-I2V-14B-720P，也就是14B大小的Wan2.2的图生视频生成5秒的720P分辨率的视频，TurboDiffusion可以可以在单张RTX 5090几乎无损端到端加速120倍：

TurboWan2.1-T2V-1.3B-480P

对于Wan2.1-T2V-1.3B-480P，也就是1.3B大小的Wan2.1的文生视频生成5秒的480P分辨率的视频，TurboDiffusion可以在单张RTX 5090几乎无损端到端加速97倍，也就是在2秒内就可以生成一个5秒的视频：

TurboWan2.1-T2V-14B-720P
