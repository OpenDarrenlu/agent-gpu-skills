# 大语言模型的反思

**作者**: NULL让智能更廉价: AI芯片软件栈与异构计算系统

**原文链接**: https://zhuanlan.zhihu.com/p/615884740

---

虽然接触nlp很早，但是因为比起cv模型的火热（17～21年），包括nlp落地更困难等因素，导致我狭隘认为cv代表了最新的ai水平，即使transformer出来之后我也认为它和cnn比是一个性价比很差的模型，我根本不会拿它落地。

其实19年就认识到cv的停滞不前（毕竟cv属于感知层），那时候在找乱找AGI的路，道理很简单，认为只要给cv感知出来的结果加上“世界知识/经验”的决策就能让cv有质变。

那时候乱七八糟资料翻了一堆，就是没有想过nlp潜力到底代表什么，直到chatGPT展现出它蕴含惊人的“世界知识/意义”，才恍然大悟，TMD我多次挂在嘴边的“人只能活在语言编制的意义之网中”，却从来没把这个道理和nlp关联起来。

语言才是认知的核心，几乎人类对“世界知识”认知都是通过语言甚至绝大部分是逻辑含糊的自然语言，离开语言人群倒退回和狼群差不多的水平；

nlp找到了一条通向AGI正确的道路，语言才是核心；通过不断地编解码训练，大模型学到了一个由语言构建的庞大的意义网络，这个意义之网几乎就是现代人类对世界的认知。对这个世界常识的认知和决策都可以通过这个“意义之网”找到匹配结果。

很多人会觉得chatGPT只是一个更方便的搜索工具，只是统计学的一个拟合，道理确实是这样的，但是谁能说人脑不也只是一个统计学到拟合器吗？人在社会构建的文化和意义之网中的决策与gpt在语言构建的世界意义之网中的拟合有什么区别呢？

单词的概念或者说意义，只能来自于某个高维空间网络中；且对单词的定义只能用差异化(几百个它不是什么)来定义它是什么，特别像高维空间中，定义一个异常值；

2. 自然语言的缺陷(数学符号语言缺陷也很大)，所以我推崇各自编程语言这类可以精准描述事物的语言来做语料

3. 是1的补充，单词的意义必须在一个网络中才行，同样的词在不同的网络中含义不一样(transformer的自注意力可以帮助单词对齐意义之网)

4. 意义之网(有了语言之后，人类强化第一个逻辑能力就是因果推断，第一个问题是为什么)，语言既是扩展我们思考的工具，也是局限我们思维的牢笼
