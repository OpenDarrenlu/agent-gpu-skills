"吸收"的作用

暂不考虑 MLA-AttnBlock 内的 Normal\rope\concat 类计算，直接从 MLA 的 Gemm 计算公式的角度切入分析“吸收”模式和“不吸收” 模式的访存。

“吸收“ 逻辑：

Q_a_proj: [q_len, H] * [H, q_lora_rank]
Q_b_proj : [q_len, q_lora] * [q_lora_rank, n *(qk_nope_dim + qk_rope_dim)]
W_uk_from_kv_b_proj ：[n, q_len, qk_nope_h] *[n, qk_nope_h, kv_lora_rank]
KV_a_proj_with_mqa : [q_len, H] * [H, kv_lora_rank + qk_rope_dim]
AttnCore :
Q * K_t: [n, q_len, kv_lora_rank + qk_rope_dim] * [kv_lora_rank + qk_rope_dim, kv_len]
AttnScore * V: [n, q_len, kv_len] * [kv_len, kv_lora_rank]
W_uv_from_kv_b_proj: [q_len, n, kv_lora]*[n, kv_lora, v_head_dim]
OutProj: [q_len, n * v_head_dim] * [n * v_head_dim, H]

“不吸收“ 逻辑：

Q_a_proj: [q_len, H] * [H, q_lora_rank]
Q_b_proj : [q_len, q_lora_rank] * [q_lora_rank, n *(qk_nope_dim + qk_rope_dim)]
KV_a_proj_with_mqa : [q_len, H] * [H, kv_lora_rank + qk_rope_dim]
K_b_proj: [q_len, kv_lora_rank] [kv_lora_rank, n * (qk_nope_dim + v_head_dim)]
AttnCore :
Q * K_t: [n, q_len, (qk_rope_dim + qk_nope_dim)] * [n, (qk_rope_dim+qk_nope_dim), kv_len]
AttnScore * V: [n, q_len, kv_len] * [n, kv_len, v_head_dim]
OutProj: [b,s,n/tp v_h] [n/tp * v_h, H]

结合上述公式推导，可以发现在最核心的 Attention kernel 计算，“吸收“模式下 K/V tensor Shape 中不携带 num_attn_heads 信息，计算逻辑转换成了 MQA 类 Attention 计算，“不吸收”模式下 K/V tensor 仍携带 num_attn_heads，考虑到 deepseek 类模型结构中 num_attn_heads 通常为 128。此时，相较于"不吸收"模式，K/V tensor 的访问规模扩大了如下倍数。如果代入 DeepseekV3 参数，则是此处 K/V 访存量相对增大了 42.6 倍.

\frac{num\_attn\_heads * (qk\_rope\_dim + qk\_nope\_dim)}{kv\_lora\_rank + qk\_nope\_dim}

结论
如 1. deepseek模型学习笔记 — DevelopNotes 0.0.1 文档 所说，根本就不叫吸收，而是应该叫结合律/运算顺序。如果想叫吸收的话，也是 W^{UK} 被吸收到 Q^C ，而不是 W^{UQ} 。
吸收模式能够大幅降低 decode attention kernel 的访存规模
可以印证 prefill 的时候其实是不要做“吸收”的，就直接按 (C^Q W^{UQ}) ({W^{UK}}^\top {C^K}^\top) 算是最优的
计算量估计

附上一个计算量估计（只算了张量的矩阵乘法计算量）的脚本，包括错误吸收的计算量增加了多少倍，以及正确吸收的计算量是不吸收的多少倍：

hidden_dim = 7168         # d
q_lora_rank = 1536        # d_C'
kv_lora_rank = 512        # d_C
num_heads = 128           # n_h
qk_nope_head_dim = 128    # d_h
qk_rope_head_dim = 64     # d_h^R
v_head_dim = 128          # d_h
 
# bsz: batch size
# q_len: query length
# kv_len: key/value length
def demo(bsz, q_len, kv_len):
    ops_original = (
        bsz * q_len * hidden_dim * q_lora_rank                        # C^Q    = X W^{DQ}
        + bsz * q_len * hidden_dim * kv_lora_rank                     # C^{KV} = X W^{DKV}
        + bsz * q_len * q_lora_rank * num_heads * qk_nope_head_dim    # Q^C    = C^Q W^{UQ}
        + bsz * q_len * q_lora_rank * num_heads * qk_rope_head_dim    # Q^R    = RoPE(C^Q W^{QR})
        + bsz * kv_len * kv_lora_rank * num_heads * qk_nope_head_dim  # K^C    = C^{KV} W^{UK}
        + bsz * q_len * hidden_dim * qk_rope_head_dim                 # K^R    = RoPE(X W^{KR})
        + bsz * kv_len * kv_lora_rank * num_heads * v_head_dim        # V^C    = C^{KV} W^{UV}
        + bsz * q_len * kv_len * num_heads * qk_nope_head_dim         # AW^C   = Q^C K^C^T
        + bsz * q_len * kv_len * num_heads * qk_rope_head_dim         # AW^R   = Q^R K^R^T
        + bsz * q_len * kv_len * num_heads * v_head_dim               # (AW^C + AW^R) V^C
        + bsz * q_len * num_heads * v_head_dim * hidden_dim           # ((AW^C + AW^R) V^C) W^O
    )
 
    # shape of C^Q    = [bsz, 1, q_len, q_lora_rank]
    # shape of W^{UQ} = [num_heads, q_lora_rank, qk_nope_head_dim]
    # shape of W^{UK} = [num_heads, kv_lora_rank, qk_nope_head_dim]
    # shape of C^{K}  = [bsz, 1, kv_len, kv_lora_rank]
 
    # (C^Q W^{UQ})(W^{UK}^T C^K^T)
    ops_k_original = (
        bsz * num_heads * q_len * q_lora_rank * qk_nope_head_dim
        + bsz * num_heads * kv_len * kv_lora_rank * qk_nope_head_dim
        + bsz * num_heads * q_len * kv_len * qk_nope_head_dim
    )
    # ((C^Q W^{UQ}) W^{UK}^T) C^K^T
    ops_k_associative = (
        bsz * num_heads * q_len * q_lora_rank * qk_nope_head_dim
        + bsz * num_heads * q_len * qk_nope_head_dim * kv_lora_rank
        + bsz * num_heads * q_len * kv_len * kv_lora_rank
    )
    # (C^Q (W^{UQ} W^{UK}^T)) C^K^T
    ops_k_wrong_associative = (
        + bsz * num_heads * q_len * q_lora_rank * kv_lora_rank
        + bsz * num_heads * q_len * kv_len * kv_lora_rank
    )
 
    # shape of AW     = [bsz, num_heads, q_len, kv_len]
    # shape of C^V    = [bsz, 1, kv_len, kv_lora_rank]
    # shape of W^{UV} = [num_heads, kv_lora_rank, v_head_dim]
    # shape of W^O    = [num_heads, v_head_dim, hidden_dim]
 
    # (AW (C^V W^{UV})) W^O
    ops_v_original = (
        bsz * num_heads * kv_len * kv_lora_rank * v_head_dim
        + bsz * num_heads * q_len * kv_len * v_head_dim
        + bsz * num_heads * q_len * v_head_dim * hidden_dim
    )
    # ((AW C^V) W^{UV}) W^O
    ops_v_associative = (
        bsz * num_heads * q_len * kv_len * kv_lora_rank
        + bsz * num_heads * q_len * kv_lora_rank * v_head_dim
        + bsz * num_heads * q_len * v_head_dim * hidden_dim
    )
    # (AW C^V) (W^{UV} W^O)
    ops_v_wrong_associative = (
        bsz * num_heads * q_len * kv_len * kv_lora_rank
        + bsz * num_heads * q_len * kv_lora_rank * hidden_dim
    )
    print("Wrongly absorb ops over correctly absorb ops:",
          (ops_k_wrong_associative + ops_v_wrong_associative) / (ops_k_associative + ops_v_associative))
 
    ops_associative = (
        ops_original
        - (ops_k_original - ops_k_associative)
        - (ops_v_original - ops_v_associative)
    )
 
    print("total absorb ops over original ops:",
          ops_associative / ops_original)
 
 
#### prefill ###
print("Prefill:")
demo(16, 1023, 1023)
 
#### decode ####
print("Decode:")
demo(16, 1, 1024)
