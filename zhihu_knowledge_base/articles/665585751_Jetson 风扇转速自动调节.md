# Jetson 风扇转速自动调节

**作者**: SnoopyBug划水划呀划~

**原文链接**: https://zhuanlan.zhihu.com/p/665585751

---

安装Jetson专用工具jetson-stats进行调节，启动jtop

$ sudo apt install python3-pip
$ sudo -H pip3 install -U jetson-stats
$ sudo jtop
开启前
点击CTRL / 按6
切换模式quiet/cool/manual：切换为cool可以自动调节风扇转速保持较低温度；manual需要人工设置转速
分别按s和e，打开Jetson Clocks和on boot开关，开机自启
重启reboot
开启后
