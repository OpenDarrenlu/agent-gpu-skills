1 波形

画波形毕竟是主打的，官方例子给的很多，直接上官网找就行。

1.1 信号命名

name中进行信号命名。

wave_name

上图的示例代码：

{ "signal" : [
    { "name" : "Signal Name" }
]}
1.2 信号类型

通过wave中不同字符表示信号类型。每个字符代表一个周期。

p ：创建时钟上升沿
n ：创建时钟下降沿
P ：创建带箭头的时钟上升沿
N ：创建带箭头的时钟下降沿
0 ：低电平
1 ：高电平
= ：默认值为2的颜色
2~9：不同颜色类型
x ：不定态
. ：延续前面的信号
z ：高阻态
u ：上拉信号（weak 1）
d ：下拉信号（weak 0）
| ：延续前面信号，带间隔
wave_types

上图的示例代码：

{ "signal" : [
  { "name": "Wave types", "wave": "pP1nN0udx=23456789.z|"}
] }
1.3 信号数据

通过data设置数据名，对应到数据值（信号类型为2-9）中。

wave_data

上图的示例代码：

{ "signal" : [
  { name: "Data",  wave: "x345x", data: "D1 D2 D3"}
] }
1.4 信号周期/相位

通过period和phase调节信号周期和相位。

wave_period&phase

上图的示例代码：

{ signal: [
  { name: "clk",  wave: "p...", period: 2  },
  { name: "clk2", wave: "p...", period: 2, phase: 0.5 },
  { name: "clk3", wave: "p.......", period: 1 },
  { name: "clk4", wave: "p." , period: 4}
]}
1.5 信号节点

通过node设置节点或者边沿的名称，范围为[a~z]，[1-9]。

wave_nodes

上图的示例代码：

{ signal: [
  { name: "Wave nodes", wave: "01..01010",  node: ".a..bcde0" },
  ]
}
1.6 信号箭头

通过edge设置加连接关系的节点或边沿名称，结合下面连接类型进行指向。

- ：水平移动箭头
| ：垂直移动箭头
~ ：形成曲线形状
/ ：添加对角线
> ：添加箭头

wave_arrow

上图的示例代码：

{ signal: [
  { name: "Wave1", wave: "01010101010",  node: ".abcdefghij"},
  { name: "Wave2", wave: "10.10.10.10",  node: ".1.2..34.56."},
  {            node: "......x...y."},
  ],
  edge: [
   "a-b","b->c","e|-3",,"h-|->5",
   "1/b","b~>2 A","2-|>d","3-x","6-y","4-|-h",
   "x<->y info"
  ]
}
1.7 信号组别
wave_group

上图的示例代码：

{signal: [
  ["Request",
    ["ctrl‘,
      {name: "wen", wave: "01.0...."},
      {name: "ren",  wave: "0...1..0"}
    ],

    {  name: "wdata", wave: "x3.x....", data: "D1"   },
  ],
   {},
  ["Response",
    {  name: "rdata", wave: "x.....4x", data: "Q2"},
  ],
     {}
]}
2 寄存器

画寄存器这个功能，官网几乎都没提到，git上介绍的很潦草，也没有一个正经的教学，找了半天才知道怎么用。

画画寄存器和指令之类的还是挺方便的。格式看起来也比较简洁。

寄存器基本参数：

bits：设置位段位宽
name：设置位段名字
type：设置位段类型（不同颜色）

显示配置参数：（其实还有些别的，但是好像没什么用）

vspace ：垂直空间，默认80。
hspace ：水平空间，默认640。
lanes ：条目数，默认2。
bits ：整体位宽
compact ：是否紧凑类型，默认为flase。
hflip ：是否水平翻转，默认为flase。
vflip ：是否垂直翻转，默认为flase。
label：加标签
