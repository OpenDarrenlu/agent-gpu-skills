FlashAttention-4 发布了，但主要是针对 Blackwell 架构做的优化，具体来说是 sm100。就算手中有 5090，也没有办法使用 TMEM 的功能。那么，如果大家特别想上手体验一下 FlashAttention-4 优化做了什么、性能提升的来源是什么，甚至想动手调一下代码，该怎么办呢？

没有 B200 也能玩 FA4

我自己在 Blackwell 架构的卡上调试算子用的是 Modal。国内据说还是可以 access 的，每个月白给 30 美刀的额度。可以在 B200、H100 上做测试，只要不是玩训练，用起来还是挺爽的，一个月根本花不完。

测试脚本分享

这里给大家分享一下我的测试脚本：https://github.com/deciding/txl/blob/main/docker/fa4_benchmark.py

使用方法超简单：

bash pip install modal modal run fa4_benchmark.py

其他什么也不用安装，基本是开箱即用，很方便。

动手调代码

我这里是直接拿的 FlashAttention-4 官方的实现。大家可以在 docker/flash_attn/flash_fwd_sm100.py 这个文件里自由发挥，说不定能调出更优更快的版本。

我用我的代码助手 Handclaw 随便调了一下，让 block_m 从128 变成 64，特意不打满 tcgen05。我们预期是性能下降的，结果显示确实下降了，但只下降了 2.5%，比我想的要少一些。
