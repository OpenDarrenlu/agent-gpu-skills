更新

直接沿着负梯度方向更新即可，在优化中可以设置步长，在机器学习里面一般称为学习率。

def fit(self):
    num_wrong = self.X.shape[0]
    while num_wrong > 0:
        num_wrong = 0
        for x, y in zip(self.X, self.y):
            if y * (np.dot(x, self.w) + self.b) <= 0:
                self.w += y * x * self.learning_rate
                self.b += y * self.learning_rate
                num_wrong += 1
        if num_wrong == 0:
            print("Fit finished!")
对偶形式

由于在更新中可见 w 为 x_iy_i 的线性组合，即 w=\alpha_ix_iy_i 。故对 w 的优化可以转换到对其对偶变量 \alpha 的优化。将 w 的形式代入损失函数即可得到对偶形式的损失函数：

L(\alpha,b)=-y_i(\alpha_jy_jx_jx_i+b)

这里求和约定的优势就体现出来了，相比矢量记法，张量记法能极大减轻公式推导的负担。其中 x_jx_i 这一项即为 x 的内积，可以提前计算出来，又称为Gram矩阵。值得注意的是，一般来说对偶问题的优化意义更加明确，往往还能减少计算量，但是对于感知机问题，其判断误分类点和原始问题都需要一个内积，并没有减少计算量，反而多了一个Gram矩阵的计算开销。

优化过程也很简单。选取一个误分类点 x_i ，则其对 w 的更新有贡献，则将其权重 \alpha_i 增加即可。 \alpha_i 越大，说明该点对最后的贡献越大。参数 b 的更新与原始问题保持一致。具体实现和原始算法区别很小，仅在条件判断和参数更新方面有不同。
