# UCSD第一学期流水账

**作者**: Peiyuan007UCSD CS PhD student

**原文链接**: https://zhuanlan.zhihu.com/p/15787343613

科研

这学期刚开学，老板就让我转Video方向。刚开始时特别搞笑，Hao给了我一个RunPod（类似国内AutoDL）账号让我平时dev用，说需要多卡就找他请示。我心想完了完了，A40开一张以上都要请示，这还怎么搞Video Generation？结果一周后才发现，真正的多卡资源在另一个集群上，完全没有卡不够用的情况。这辈子从来没见过这么好的猪肉。

本以为第一学期可以慢慢开始科研，结果直接进入爆肝模式。边学Diffusion边写代码，做着做着发现Diffusion的开源框架远不如LLM那样“百花齐放”（可能因为之前Diffusion模型规模较小，用Diffusers就够了）。但参考LLM的经验，训练（如LLama-factory、xtuner、）和推理（如vLLM、SGLang, LMDeploy）都能火(卷)出很多框架。如果未来Video Generation能和LLM齐头并进，那现在搞Video Diffusion的开源框架还有很大潜力（尤其是未来Video模型规模会越来越大）。于是拉着几个intern火速开搞，边上课边爆肝， 这也导致基本没怎么做paper导向的科研。

两周前为了蹭Sora的热度，赶着把FastVideo放了出去，热度确实蹭到了，但因为团队人手和时间有限，大家之前也没怎么做过Diffusion，目前代码质量我个人还不太满意。接下来会持续开发和重构，也欢迎对这个方向感兴趣的同学加入。目前也还在思考Diffusion serving和LLM prefill有没有显著的差异。




上课

这学期选课可以说是大失败。我选了两门课：240D Specialized Accelerator for Deep Learning 和 260 Parallel Computing。虽然260的workload是公认的大，但我觉得还好，三个PA每个基本一两天就能搞定。反倒是240D踩了大雷——原以为是水水的读paper课，结果发现每周要读四篇paper+写reflection。如果是AI的故事会还好，但偏偏全是Hardware，我完全看不懂。更折磨的是project：直接甩给我们一个毫无注释的verilog systolic array代码，还要复现Planaria，结果发现连ISA和compiler都得改。本以为找了个做compiler的大佬队友能摆烂，结果发现大佬也在摆……最终没做完，幸好TA心软，给了个还不错的分数。

总结教训：下次不能选两门系统课，最好一门系统课配一门数学或算法相关的课。虽然240D这门课确实让我学到了一些早期Deep Learning硬件的奇葩设计（虽然最后发现还是systolic array最香）。







实习




这点要给Hao打满分。他是一个要求高、hands-on、自身能力非常强、又非常支持学生的老师。有时还和学生一起看代码甚至debug。而在支持学生找实习方面更是用心，每个PhD都会帮忙用connection推荐实习。有这样的导师真的很幸运。







生活

第一学期真的很忙， 刚来SD时认识了很多同一届的master和PhD学生，一开始大家玩得挺好。但后来项目一忙，就完全没时间connect了，还好还有飞机上认识的朋友周末有时候会一起做饭，感觉这一点以后确实需要更好地平衡一下。买车， SSN这些全都没搞, SD地图开疆进度基本为0， 这里不得不说我去了三次SSA结果SSN都没有办下来， 第一次学校没有激活i20身份， 第二次SSA改政策变成不能walkin, 第三次遇到一个很mean的officer非要说offer letter上的签名不能是机器打印的（问了三个其他人都没有遇到这个问题。


作为一个从新加坡来到SD的新人，刚来的第一个月，每天早上出门都会感慨一句：SD的天气是真的好，满满的感恩之情。治安也是above my expectation了——经常晚上10点一个人走路回宿舍，基本没有不安全感。距离学校开车20分钟就是城市（SD为美国第八大城），基本没有大家经常吐槽的美国学校那种“村”的感觉。再加上海真的好看，周围玩儿的地方也不少，真心觉得UCSD算是美国少有的，把安全、好天气、不村这三点全占了的全能战士学校！
