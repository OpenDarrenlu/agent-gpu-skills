leetcode-cn.com/problems/median-of-two-sorted-arrays/

难度：Hard

给定两个大小分别为m和n的正序（从小到大）数组 nums1和 nums2。请你找出并返回这两个正序数组的中位数。要求O（log（m+n））。

输入：nums1 = [1,3], nums2 = [2]
输出：2.00000
解释：合并数组 = [1,2,3] ，中位数 2

输入：nums1 = [1,2], nums2 = [3,4]
输出：2.50000
解释：合并数组 = [1,2,3,4] ，中位数(2 + 3) / 2 = 2.5

思路，假设nums1[ ]有m个数，nums2[ ]有n个数。首先分类讨论：

1、m+n为奇数，则中位数为合并后第(m+n+1)/2个数；

2、m+n为偶数，则中位数为第(m+n)/2个数与第(m+n)/2+1个数的平均值。

问题转化为“寻找两个数组的第k小的数”，定义为函数int findk( )。

使用二分查找来达到O(log(m+n))复杂度。

p和q为两个数组的当前下标，起始为0。

根据二分查找的思路，每次下标移动k/2。

p2=p1+k/2;
q2=q1+k/2;

如果p2超过nums1数组的边界，则用m代替，q2同理。

p2=min(p1+k/2，m)

如果下一步从p2或q2开始，则参与比较的是左边的数（p2-1）或（q2-1）

newp=min(p+k/2-1,m-1)
newq=min(q+k/2-1,n-1)

如果有nums1[newp]<nums2[newq]，则nums1[p,newp]是最小的几个数，可以跳过，更新对应的k和p。

k2=k1-（newp-p+1)
p2=newp+1

另一种情况nums1[newp]≥nums2[newq]，更新对应的k和p。

k2=k1-（newq-q+1)
q2=newq+1

如果p达到了m，且k＞1，结果在nums2[ ]

if（p==m）return nums2[q+k-1]

另一种情况

if（q==n）return nums2[p+k-1]

如果k减小到1，则收敛至结果

if（k==1）return min(nums[p]，nums[q])

最终代码

class Solution {
public:
    double findMedianSortedArrays(vector<int>& nums1, vector<int>& nums2) {
        int m=nums1.size();
        int n=nums2.size();
        if((m+n)%2==1){
            return findk(nums1,nums2,(m+n+1)/2);
        }
        else{
            return (findk(nums1,nums2,(m+n)/2)+findk(nums1,nums2,(m+n)/2+1))/2.0;
        }
    }

    int findk(vector<int>& nums1,vector<int>& nums2,int k){
        int m=nums1.size();
        int n=nums2.size();
        int p=0,q=0;

        while(true){
            if(p==m) return nums2[q+k-1];
            if(q==n) return nums1[p+k-1];
            if(k==1) return min(nums1[p],nums2[q]);

            int newp=min(m-1,p+k/2-1);
            int newq=min(n-1,q+k/2-1);

            if(nums1[newp]<nums2[newq]){
                k=k-(newp-p+1);
                p=newp+1;
            }
            else{
                k=k-(newq-q+1);
                q=newq+1;
            }
        }
    }
};
