# KD research的小总结

**作者**: 东川路第一伊蕾娜​Open and open again

**原文链接**: https://zhuanlan.zhihu.com/p/1948334729662280166

---

1.MiniLLM: Knowledge Distillation of Large Language Models：

首次提出了反向KL散度，优化学生模型聚焦教师模型的高概率区域，但仍然是off-policy水平。

2.On‑Policy Distillation of Language Models: Learning from Self‑Generated Mistakes

（GKD）

提出了on-policy KD概念，证明on-policy KD性能最好，同时和RL首次结合，说明了KD和RL结合的trade-off问题。但是选择的是小模型，停留在full-vocab KD的非工业化阶段。

3.KDRL: Post-Training Reasoning LLMs via Unified Knowledge Distillation and Reinforcement LearningKDRL: Post-Training Reasoning LLMs via Unified Knowledge Distillation and Reinforcement Learning

提出了GRPO+KD结合的方式，创新性的结合KL k2散度（维持梯度无偏）和反向KL（类似奖励），把蒸馏结合到RL领域。同时也做了很多trick：1.随step增加，逐步从KD->RL，缓解了KD和RL的trade-off；2.只对错误的response做KD。证明了：1.predict token logits KD+RL比top KKD+RL更有竞争力，符合工业化；2.KD做reward shape面对崩溃，采用A+B的结合方式更为稳健。

4.Qwen 3 tech report：

提出了adaptive thinking 模型做KD的方式：先off policy学会/think，/no_think的格式，再on policy学会两种模式。

5.KAT-V1: Kwai-AutoThink Technical Report：

提出了KD+MTP的方式，构建ULD Loss，提升了性能和效率，但可惜没有写具体的测试结果。

6.Gemma 3 Technical Report：

提出了KD 前256的teacher分布的方法，相比于以前full-vocab，效率获得极大的提升，从非工业化到足够工业化。
