# 如何用光学解决最速降线问题

**作者**: 马同学​数学话题下的优秀答主

**原文链接**: https://zhuanlan.zhihu.com/p/10701324882

---

中国古建筑的屋顶常被设计成弧线，如下图左侧所示，这一设计源于古代设计者对雨水排放效率的认识。正如先秦科技著作《考工记》所记录的："上尊而宇卑，则吐水疾而溜远"，如下图右侧所示，意指上半部分坡度大，下半部分坡度小，从而使雨水能迅速排出并流得更远。

相比直线屋顶，曲线屋顶能更快地排出雨水，其中蕴含的数学原理可以参看下面的例题。考虑竖直平面上的A 、B 两点，其中A 点位于B 点的左上方，如下图所示。想象仅受重力作用的小球从A 点出发，沿不同的曲线轨道滑向B 点。由于各曲线的长度、形状各异，小球在不同路径上的速度变化不同，导致到达B 点的时间有所差异。




其中存在一条特殊的曲线，小球沿该曲线从A 点滑到B 点的耗时最短，该曲线被称为最速降线（Brachistochrone Curve），上图中的红色曲线就是这里的最速降线。请求出最速降线。

解.（1）许多数学家都曾探索过最速降线问题，其中瑞士数学家约翰·伯努利（ohann Bernoulli，见下图左侧）巧妙地利用光学原理找到了答案。他设想A 、B 两点浸没在密度渐变的液体中（如下图右侧所示，这里用渐变色表示密度的变化），根据光学中的费马原理 ，光从A 点出发会不断发生折射，最终沿耗时最短的路径到达B 点，这条路径即为A 、B 两点间的最速降线。



这里解释下约翰·伯努利的思路：小球从A 点滑向B 点时，在重力作用下其速度不断变化，如下图左侧所示。类似的，光在密度不同的溶液中传播速度也会变化。通过调整溶液密度，使光速和小球速度同步变化，如下图右侧所示，即可将力学中的最速降线问题转为光学中的折射问题。

（2）下面开始具体的求解，先计算球速以及光速。如下图左侧所示，以A 点为原点建立坐标系。质量为m 的小球从A 点出发滑下y_0 高度后，根据能量守恒定律，其获得的重力势能mgy_0 会全部转化为动能\frac{1}{2}mv^2 ，即有mgy_0=\frac{1}{2}mv^2 ，因此可推出小球在该点的速度v=\sqrt{2gy_0} 。根据（1）中的分析，在对应高度的溶液处，也有光速v=\sqrt{2gy_0} ，如下图右侧所示。

接下来为了简化光在密度渐变液体中的传播分析，我们将液体水平分为n 层，如下图所示。假设每层液体的密度保持恒定，那么可知，

光在每层内直线传播且速度恒定
光在层间交界处发生折射，根据光学中的斯涅尔定律，所以有：\frac{\sin\theta_0}{v_0}=\frac{\sin\theta_1}{v_1}=\frac{\sin\theta_2}{v_2}=\frac{\sin\theta_3}{v_3}
光从A 点出发，经层层折射后，沿红色折线段传播到B 点。该折线段耗时最短，也就是此时的最速降线

当液体水平分为无限层（n\to\infty ）时，我们就得到了密度渐变的液体。在这种情况下，光线从A 点出发经历无数次微小折射后，会形成一条从A 点到B 点的平滑曲线路径，这就是本题中要求的最速降线，如下图所示。这条曲线上每一点的切线与竖直线的夹角即为入射角（这一点还是符合直觉的，在约翰·伯努利的时代，直觉往往指导着科学研究）。作为示意，这里作出了几个点及其入射角，并根据（1）中的分析计算出该点的光速，结合上斯涅尔定律，因此会有\frac{\sin \theta_0}{v_0}=\frac{\sin \theta_1}{v_1}=\frac{\sin\theta_2}{v_2} ，代入光速并引入某常数C_1 可得\frac{\sin \theta_0}{\sqrt{2gy_0}}=\frac{\sin \theta_1}{\sqrt{2gy_1}}=\frac{\sin\theta_2}{\sqrt{2gy_2}}=C_1 。




设上述平滑曲线对应函数y=f(x) ，并将坐标系以及函数f(x) 的图像上下翻转，采用常规的直角坐标系表示，如下图所示。那么对于任意一点x 对应的函数值y=f(x) ，都有\frac{\sin\theta}{\sqrt{2gy}}=C_1 。这里还标出了和\theta 互余的\theta' 角。

根据之前的学习可知\tan(\theta')=y' ，而\theta+\theta'=\frac{\pi}{2} ，所以有\cot\theta=y' 。结合上三角函数的公式1+\cot^2\theta=\frac{1}{\sin^2\theta} ，可推出：

\sin\theta=\frac{1}{\sqrt{1+\cot^2\theta}}=\frac{1}{\sqrt{1+y'^2}},\quad 0 < \theta < \frac{\pi}{2}

将上式代入\frac{\sin\theta}{\sqrt{2gy}}=C_1 ，就得到了一个关于函数y=f(x) 的一阶微分方程：

\begin{aligned} \frac{\sin\theta}{\sqrt{2gy}}=C_1 &\implies \frac{1}{\sqrt{1+y'^2}}\cdot\frac{1}{\sqrt{2gy}}=C_1\implies \sqrt{y(1+y'^2)}=\frac{1}{\sqrt{2g}C_1}\\ &\implies y(1+y'^2)=\frac{1}{2gC_1^2}\implies y(1+y'^2)=C_2,\quad C_2=\frac{1}{2gC_1^2} \end{aligned}

（3）解微分方程y(1+y'^2)=C_2 。整理之后变化为可分离变量的微分方程：




设y=C_2\sin^2\varphi ，所以\mathrm{d}y=2C_2\sin\varphi\cos\varphi\mathrm{d}\varphi ，代入上式并运用半角公式\sin\frac{\theta}{2}=\pm\sqrt{\frac{1-\cos\theta}{2}} ，可得：

\begin{aligned} x &=\int\sqrt{\frac{y}{C_2-y}}\mathrm{d}y=\int\sqrt{\frac{C_2\sin^2\varphi}{C_2-C_2\sin^2\varphi}}\cdot 2C_2\sin\varphi\cos\varphi\mathrm{d}\varphi\\ &=\int\frac{\sin\varphi}{\cos\varphi}\cdot 2C_2\sin\varphi\cos\varphi\mathrm{d}\varphi=C_2\int 2\sin^2\varphi\mathrm{d}\varphi\\ &=C_2\int(1-\cos 2\varphi)\mathrm{d}\varphi=\frac{C_2}{2}(2\varphi-\sin 2\varphi)+C_3 \end{aligned}\\

这样我们就得到了函数y=f(x) 的参数方程，或者说得到了最速降线的参数方程：

\begin{cases} x=\displaystyle\frac{C_2}{2}(2\varphi-\sin 2\varphi)+C_3\\ y=C_2\sin^2\varphi \end{cases}\\

这里希望\varphi=0 时有x=0 和y=0 （或者说\varphi=0 时，最速降线上对应的点在原点处），所以C_3=0 。再运用半角公式\sin\frac{\theta}{2}=\pm\sqrt{\frac{1-\cos\theta}{2}} ，上述参数方程可以改写如下：

\begin{cases} x=\displaystyle\frac{C_2}{2}(2\varphi-\sin 2\varphi)\\ y=\displaystyle C_2\sin^2\varphi=\frac{C_2}{2}(1-\cos 2\varphi) \end{cases}\\

令\displaystyle a=\frac{C_2}{2} 以及t=2\varphi ，上述参数方程可以改写如下，这是更常见的最速降线的参数方程：

\begin{cases} x=a(t-\sin t)\\ y(t)=a(1-\cos t) \end{cases}\\

比对一下可知，最速降线也是之前介绍过的摆线。
