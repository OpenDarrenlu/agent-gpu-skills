环境配置

cuTile 主要依赖的 CUDA13.1 的 tileiras，所以需要安装一下 CUDA13.1

但是现有 jetpack 目前驱动只支持到580以及 CUDA13.0，所以第一时间想到的就是 docker

$ nvidia-container-cli info

Architecture:   11.0

直接启动 docker 会出现 driver 不够高的问题

docker: Error response from daemon: failed to create task for container: failed to create shim task: 
OCI runtime create failed: failed to create NVIDIA Container Runtime: 
failed to construct OCI spec modifier: requirements not met: cuda>=13.1||brand=unknown&&driver>=535&&driver<536||brand=

所以启动 docker 的时候，需要注意下加上-e NVIDIA_DISABLE_REQUIRE=1即可（管他三七二十一，能跑才是第一）

docker run -it \
    --gpus all \
    -v 你的挂载目录:/workspace \
    -w /workspace \
    -e NVIDIA_DISABLE_REQUIRE=1 \
    -e http_proxy=$http_proxy \
    -e https_proxy=$https_proxy \
    -e no_proxy=$no_proxy \
    nvcr.io/nvidia/cuda:13.1.0-devel-ubuntu24.04

拉下来这个 docker 是个很纯净的环境，安装一下基础的环境

apt update
apt install -y     ca-certificates     curl     wget     git     vim     build-essential     software-properties-common
apt install -y python3 python3-pip python3-venv
apt install -y python3-venv python3-full

然后可以在 workspace下面 clone cuTile 的源码，由于是个 docker，也不用管环境的问题了，直接安装

cd /workspace/cutile-python
pip install cuda-tile --break-system-package -i https://mirrors.aliyun.com/pypi/simple/
pip install -r test/requirements.txt --break-system-package -i https://mirrors.aliyun.com/pypi/simple/
