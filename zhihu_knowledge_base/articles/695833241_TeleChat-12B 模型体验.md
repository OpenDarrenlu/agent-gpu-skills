背景

之前评测过的由中电信人工智能科技有限公司推出的星辰语义大模型 TeleChat 最近推出了更大的 12B 版本。相比于 7B 模型，12B 模型使用的数据量多了一倍（1.5T -> 3T），模型结构上也有更新，在绝大部分主流的评测指标上都有较大的提升。和 7B 模型同样，本次开源的 12B 模型也同时包含 FP16、int8 和 int4 版本和推理部署代码。

代码浏览

和 7B 模型同样，本次 12B 模型也是开源在 GitHub 和 Hugging Face 上（一个小小建议：作为以国产为特色的模型，可以考虑把权重也开源在一些“国产 HF”上），代码结构也和 7B 基本一致。

7B 模型开源时的一些问题，例如 GitHub 和 HF 的 README 内容不一致，在 12B 开源时已经修复，但另一个问题，GitHub 仓库大小为 0 的空权重文件，仍然存在，期待开发者们的修复。

进一步体验

找了一台带 GPU 的机器继续测试。

数学能力相比 7B 有提升，7B 模型无法回答 “4减4等于几”，但 12B 模型回答正确。不过对于 6 除以 3 这样的简单除法，模型也强行使用全套短除法，用户体验有可改进的空间，同时短除法过程是否正确也值得斟酌（“被除数变为了3，所以除数变成了3”）。
Q: 4减4等于几
A: 4减4的结果是0。
Q: 6除以3等于几
A: 6除以3的结果是2。
解释如下：
当我们进行除法运算时，可以使用长除法或短除法。在这个问题中，我们使用短除法进行计算。短除法是将被除数（这里是6）分成若干个相等的部分，然后找出每部分可以包含的最大数（这里是3）， 然后从被除数的左边开始，每隔一个数取一个部分，将剩下的数作为答案。
具体步骤如下：
将6分成3个相等的部分（因为3*2=6）。
然后取出每部分里面的最大数（这里是3），即6中可以包含的最大数。
将6减去取出的最大数（3），即6-3=3。
被除数变为了3，所以除数变成了3。
判断3是否可以整除6。由于3乘以2等于6，所以3可以整除6。
因为商是2，所以余数是0。
因此，6除以3的结果是2。

Q: 鲁迅和周树人是什么关系
A: 鲁迅和周树人是同一个人。

总结

正如 TeleChat 7B 推出时的评价，中国电信作为国企在大模型这样的前沿技术领域能够有较大的动作并且开源，是优秀的技术水平以及高度社会责任心的表现。TeleChat 12B 模型相比于 7B，在绝大多数主流评测指标上都有较大提升，实测在绝大多数问题上也有更好的表现，对国产大模型感兴趣的小伙伴们可以尝试。
