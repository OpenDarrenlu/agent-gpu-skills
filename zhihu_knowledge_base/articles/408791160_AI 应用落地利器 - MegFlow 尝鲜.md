MegFlow 简介
MegFlow 是一个面向视觉应用的流式计算框架, 目标是简单、高性能、帮助机器学习应用快速落地。

经历过AI应用上线的同学肯定深有体会，要让一个模型从丹炉走出来到搭建应用 Flow 所涉及的东西又多又杂，技术储备不足的小厂很难在短时间内搞定所有的事情。旷视最近开源的 MegFlow 正好针对我们的痛点开出了一剂良药。

15 分钟运行 Hello World
准备源码

从 GitHub 我们可以拉取到 MegFlow 的源码。

git clone https://github.com/MegEngine/MegFlow.git
安装 Rust

MegFlow 的主体部分使用 Rust 编写，我们需要先安装 Rust。

curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
安装 Python3.x

MegFlow 支持 >=Python3.6 <=Python3.8，这里我们使用 conda 直接创建一个 3.8 的环境

conda create -n megflow python=3.8
conda activate megflow
编译 MegFlow

运行下面的命令编译 MegFlow

sudo apt install clang
cd MegFlow
cargo build

看到类似下面的输出就说明编译完成了

$ cargo build                                            
   Compiling pyo3 v0.14.1
   Compiling numpy v0.14.1
   Compiling flow-rs v0.1.0 (/mnt/home-nas/work/repo/MegFlow/flow-rs)
   Compiling flow-plugins v0.1.0 (/mnt/home-nas/work/repo/MegFlow/flow-plugins)
    Finished dev [unoptimized + debuginfo] target(s) in 10.06s
安装 MegFlow Python Binding

MegFlow 提供了易用的 Python 接口，使用下面的命令安装

cd flow-python
python setup.py install

看到类似下面的输出就说明安装完成了

Processing dependencies for pyflow==0.1.0
Finished processing dependencies for pyflow==0.1.0
运行 Hello World

MegFlow 自带了一个本地测试，用于检查编译安装是否成功，使用下面的命令运行

cd examples
cargo run --example run_with_plugins -- -p logical_test
