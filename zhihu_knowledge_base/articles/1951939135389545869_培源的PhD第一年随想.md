人在SD， 刚下飞机（doge), ICLR赶不上了，新学期还没开始， 干脆来写一点东西总结一下自己的PhD第一年。

实习

暑假在湾区字节实习，当时能拿到这个实习基本是靠我老板 Hao 全程给力，帮我推了好几个地方，最后选了字节。现在回头看，确实很适合我：既能延续我在学校做的 video sparse attention 方向，又能拿到很多算力资源（老实说，很难想象还有哪家公司会给一名 PhD 一年级的实习生这么多卡），scope对实习生来说也大得惊人。

我被 Training Infra 组 host，大部分时间在做 Seedance 的算法相关工作，正好卡在我的舒适区：infra 水平在我周围那些10x engineer里算不上出彩，但对算法理解比较深入；加上我的工程能力（和肝），让我在做算法迭代时比较敏捷。总结一句就是：比我会写代码的没我帅，比我帅的没我会写代码？顺利的话，Seedance 下一版应该能留下点我的痕迹 ，且听龙吟[doge]！

团队氛围真的很好，mentor 和同事都超级强，peer 里更是卧虎藏龙，周围不少四大的 PhD intern，我感觉基本上AI的任何问题在距离我工位20步之内都能找到人聊。其中某位伯克利帅哥我觉得更是未来的超级新星，必须抱紧大腿（笑）。大家都很愿意帮忙，印象最深的是我刚开始接触 torch compile 的 dynamic shape，不熟悉还一连踩了两个 torch compile的bug，工程团队的 leader 基本连续一周每天都抽时间和我开短会，最后一步步把坑填平。

实习生活本身也很不错。食堂我感觉是湾区大厂里最好吃的，不过中餐质量相比刚去得时候有点下降。公司每周五会发喜茶，算是鼓励大家社交的小仪式；还会给实习生送水杯、衣服、书包之类的礼物，细节上能感受到被照顾。实习薪资水平据我了解也是湾区大厂里除了 OpenAI 之外的 top，大于meta/nvidia。

字节San Jose工区

回学校：

虽然在字节、湾区的实习体验很好，但回到 UCSD 还是有一种“回家”的踏实感。圣地亚哥的天气真的天下第一：不像湾区那样干巴巴、绿色稀薄，这里有海滩、有树荫，科研压力大时走一圈就能缓过来。氛围也很不一样——没有湾区那种同质化的语境，大家天天只谈包裹和融资。回校后正好和来 UCSD 暑研的上交学弟聊天，聊到他们这一届的去向。我个人的感受是，UCSD就是“最好的地方 + 最好的学校”的组合——利益相关暴论一句：如果拿到的不是五大的 offer，选 UCSD 绝对不会是一个 wrong choice。所以当听说他们这届放弃保研的人只有几个是，我既惊讶也觉得有点可惜。在我这段时间的经历里，虽然美国的政治环境让人担心，但 PhD 依然非常值得来，资源、平台和回报都足够支撑你把一件事做深做透, 小扎曾经说过，The biggest risk is not taking any risk. In a world that’s changing really quickly, the only strategy that is guaranteed to fail is not taking risks。

说到这里，也顺便打个广告，欢迎大家申请 UCSD/HDSI。在这里，生活层面的担忧可以降到最低，精力更多可以放在科研或者兴趣爱好上：每月三次组会午饭可以报销，每个 PhD 有 2500 美金的办公设备报销额度，RA 工资 4000 美金，La Jolla 社区干净安全，中餐也不错。

UCSD MLSy







整体下来，UCSD 给我的感受就是一种能让人心安的科研环境：生活条件稳定，社区氛围充实，组里的节奏也清晰。对我来说，这种踏实和长远的安全感，才是第一年最大的收获。

Research：

一直在一边科研发paper一边做fastvideo, 分为三个cycle, 第一个cycle去年年底做了hunyuan的distillation涨了很多star, 第二个cycle花了好几个月， 还写了design doc去做inference, 结果反响平平， 第三个cycle又回过头去做distillation果然又涨了star. 回顾过去一年， 现在fastvideo的其实还是没有达到当初的雄心的（做video届的vllm[狗头])， repo里面也有很多technical debt， 不过好歹取得了一些成就: issue数量来到了300个， 渐渐有了些external PR, 很多大厂的工作开始使用fastvideo (字节的DanceGRPO, 腾讯的SRPO), 甚至最开始的几个核心开发者都因为工业界package太大而离开了fastvideo，不申phd了[狗头]。



FastVideo

科研方向上，我还是坚定押视频里的稀疏注意力。视频是 compute-bound，本身也很稀疏，不需要搞 trick 去拼 matmul (NSA). 但最大的问题是没有 benchmark，大厂内部用人工测评，学术界几乎无从下手。于是写 kernel 的人更愿意做 LLM，因为 benchmark 成熟、迭代快；CV 背景的人则喜欢做diffusion distillation或者在 ImageNet 上刷 FID。视频这边要是能有人搭起一个像 ImageNet 一样的 dataset，再配上一套公认的 metric，让大家能比着刷榜，那绝对是造福众生。

展望未来就纯属吹水了。我几乎确定 Video World Model 会在 2026/2027 成为大主旋律。湾区已经一堆创业公司在搞了，别看twitter demo真的很炫酷， 但是从技术上来说，基本套路就是在 Wan 上跑 Self-Forcing，收集点数据就能去融资。Self-Forcing 开源属实造福众生，不光创业公司，学术界也跟着捡便宜（respect）。人类真的很容易被impress, 上一次这种时候是ChatGPT, 感觉属于video world model差不多的时刻也快了。）。我的几个小预测：第一，sparse attention/linear attention 的结构改进会变得更重要。LLM 里 100K 上下文覆盖 99% 的场景，但在 world model 里，100K context 可能只能稳稳生成5秒高清视频，长时建模还是大坑。第二，更高压缩比的 VAE 可能是出路，但坑还是很多，至少 wan 5B 和 1.3B 效果差不了多少已经说明问题了。第三，video world model 会慢慢吞并周边赛道，很多做 3D 的人会转过来，以及最后和 VLA 汇流成一个更大的统一方向。顺带安利一篇长文：Xun Huang 的 world model 博客，很强。


From https://www.xunhuang.me/blogs/world_model.html

另外再说一下World Model这个词： 我觉得既然LLM能造Foundation Model这个词， 那Video Gen也能用Video World Model这个词， 先把牛皮吹出去Video从业者就能吃到更多资源[doge]

最后反思一下工作模式。我觉得自己挺能抗压的，但心态其实不算好。实验一跑崩，整个人就泄了气，周末也不想出门，早上不想健身——明明这些东西本来应该是用来抗压的，结果被我倒过来当成了科研结果的附属品。作息也相当混乱，黑眼圈越来越重，以前的兴趣爱好也有些中断。信息获取也变了。以前每天早上刷 arXiv，现在基本不行了，信噪比太低，刷完也就图一乐，真能帮到科研的没几个。现在更多是靠交流，或者Flash Linear Attention交流群[狗头], 也不知道这样是好是坏。

总结下来，给自己的要求其实很简单：心态别太容易被实验绑架，多写点code，继续在稀疏注意力这条线上深挖。评测和数据集这块要是有机会，也希望能贡献点东西，只要一直动手，总能做出点东西。
